# 🚲 Bike Sales Dashboard using Microsoft Excel

## 📌 Project Overview

This project analyzes customer bike purchasing behavior using Microsoft Excel. The objective was to identify patterns and factors that influence whether a customer purchases a bike.

The dashboard was built entirely in Excel using Pivot Tables, Pivot Charts, Slicers, and Data Visualization techniques.

---

## 🎯 Business Problem

A bike company wants to understand which customer segments are more likely to purchase bikes. By analyzing customer demographics and purchasing behavior, the company can make better marketing and sales decisions.

---

## 🛠 Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Cleaning
- Dashboard Design

---

## 📊 Dashboard Features

### Average Income per Purchase
Analyzes the relationship between customer income and bike purchases across genders.

### Purchase Behavior by Age Group
Compares bike purchases among:
- Young Adults
- Middle-Aged Customers
- Older Customers

### Purchase Behavior by Commute Distance
Examines how commute distance impacts purchasing decisions.

### Interactive Filters
Users can dynamically filter the dashboard by:
- Marital Status
- Occupation
- Number of Children
- Education Level

## 🔍 Key Insights

- Customers with higher average income showed a greater tendency to purchase bikes.
- Middle-aged customers represented the largest customer segment.
- Customers with shorter commute distances showed stronger purchasing activity.
- Occupation, education level, and family status influenced purchasing behavior.
- Interactive slicers allow deeper exploration of customer segments.
